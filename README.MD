## Simple start

1. **`npm run setup`** at the root
2. **`npm run dev`** at the root
3. open **`http://localhost:9000/`**

OR

1. **`npm install`** at the root
2. **`npx simple-git-hooks`** at the root
3. **`npm run dev`** at the root
4. open **`http://localhost:9000/`**
